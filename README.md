A **citizen-centric web portal** designed for **real-time grievance reporting, issue tracking, and civic feedback collection** to bridge the gap between citizens and municipal authorities.

---

## ‚ú® Key Features & Functionality

CivicPulse provides a complete ecosystem for efficient city management:

* **Citizen-Centric Grievance Submission:** Users can submit complaints (sanitation, street lights, water, roads, etc.) complete with **images and location data** for accurate reporting.
* **Dynamic Role-Based Access:** The platform securely segregates access for **Citizen**, **Admin**, and **Department Officer** roles.
* **Real-time Tracking:** Citizens can monitor the status of their complaints in real-time (**Pending** / **In Progress** / **Resolved**).
* **Centralized Grievance Management (Admin):** Administrators utilize a powerful dashboard to view, categorize, **assign issues to relevant officers**, and set **Service-Level Agreements (SLAs)**.
* **Officer Workflow Automation:** Department Officers receive assigned tasks, **mark progress**, and provide final resolution notes and **verification images**.
* **Feedback & Rating System:** Citizens can **rate resolved issues** and have the critical option to **reopen** a complaint if unsatisfied with the resolution.
* **Actionable Analytics & Reporting:** Dashboards provide visual data on complaint distribution (category-wise, zone-wise) and **SLA performance tracking** for continuous improvement.

## üíª Tech Stack

This project is built using a modern, robust, and scalable technology stack.

| Layer | Tools/Software | Highlights |
| :--- | :--- | :--- |
| **Frontend** | **React Js** | Modern JavaScript library for building fast, component-based UIs. |
| **Styling/UX** | **Tailwind CSS, G-SAP** | Utility-first CSS framework for rapid styling; G-SAP for smooth, performant animations. |
| **Backend** | **Spring Boot, Spring Security** | Provides robust, production-ready REST APIs and handles JWT-based authentication. |
| **Database** | **MySQL** with JPA/Hibernate | Reliable relational database for persistent storage. |
| **Charts/Analytics** | **Chart.js / ApexCharts / Recharts** | Libraries for generating dynamic and insightful visual reports. |
| **DevOps & Tools** | Postman, GitHub | Used for API testing and version control. |
